## 🍦 소개

아래의 5가지 JavaScript 기능이 있는 [자기소개 웹페이지](https://shson610.github.io/Mid-Term-Project/)입니다.

1. 상단 이동 스크롤 버튼
2. 타이핑 효과
3. 이미지 슬라이드
4. 모달창
5. 진행바 (Progress bar)

## ✨ 구현한 기능

### 1. 상단 이동 스크롤 버튼
오른쪽 하단의 위쪽 화살표 아이콘

### 2. 타이핑 효과
첫 화면 인사말

### 3. 이미지 슬라이드
About 좌측 화면에 인물 이미지

### 4. 모달창
About 우측 하단에 Career, MBTI

### 5. 진행바
탑메뉴 하단에 progress bar
